# Color Consistency Update - Dark Green Theme

## Overview

Replaced all light green colors (#43e97b, #38f9d7) with dark green colors (#10b981, #059669) across the entire frontend application for better visual consistency and readability.

## Changes Made

### Files Updated

1. **frontend/app/(tabs)/control.tsx**
   - Open Curtain button gradient
   - Already using dark green for Auto Mode indicators

2. **frontend/app/(tabs)/dashboard.tsx**
   - Curtain position progress bar gradient
   - Mode status card gradient
   - Quick action button (Open) background and shadow

3. **frontend/app/(tabs)/notifications.tsx**
   - Manual control notification type gradient
   - Empty state icon gradient
   - Alert type icon gradient (Manual Control)
   - Empty icon shadow color

4. **frontend/app/(tabs)/profile.tsx**
   - Auto Mode settings button gradient
   - Text Size settings button gradient
   - Privacy Policy button gradient
   - Export Data button gradient

5. **frontend/app/(tabs)/history.tsx**
   - Average Humidity stat card gradient

## Color Scheme

### Before (Light Green)
- Primary: #43e97b
- Secondary: #38f9d7

### After (Dark Green)
- Primary: #10b981
- Secondary: #059669

## Benefits

1. **Better Readability**: Dark green provides better contrast and is easier to read
2. **Visual Consistency**: All green elements now use the same color palette
3. **Professional Look**: Dark green appears more polished and professional
4. **Accessibility**: Improved color contrast ratios for better accessibility

## Testing

All files have been checked for TypeScript errors and no diagnostics were found.
